# AION Story Engine Assets

This directory contains example assets and templates for the AION Story Engine.

## Asset Types

### Characters
Example character definitions with traits, goals, and behaviors.

### Locations
Reusable environment templates with descriptive metadata.

### Items
Prop and object definitions with interaction capabilities.

### Themes
Narrative templates and style guides for different story types.

## Usage

Assets in this directory can be referenced and imported into story projects.
They serve as starting points and building blocks for narrative creation.
